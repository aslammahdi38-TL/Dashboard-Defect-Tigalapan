Hubungkan dashboard ini ke Google Sheets Anda

🔗 Masukkan URL Google Sheets Anda

Dashboard ini akan otomatis tarik data dari Google Sheets setiap kali dibuka atau di-refresh.

1
Buka Google Sheets Anda Pastikan sheet berisi data dengan kolom: PO, Bulan, Vendor, Kode, Produksi, Reject, Bahan, Bolong, BS, Noda, Oli, DblBenang, Kotor, Bowing, Melintir, Mat, Proses
2
Publish ke web File → Share → Publish to web → Pilih sheet → CSV → Publish → Copy URL
3
Paste URL di bawah ini URL formatnya seperti: https://docs.google.com/spreadsheets/d/.../export?format=csv&gid=...

📋 Format Kolom Google Sheets yang Diharapkan
Baris 1 = header · Baris 2 dst = data · Kolom harus sesuai urutan
